
# CSS Recomendaciones de estilos:
+ __Font-size h1:__ entre 32 y 90px para el texto H1
+ __Font-size h2:__ entre 25 y 32px para el texto H2
+ __Font-size h3:__ entre 20 y 22px para el texto H3
+ __Font-size p:__ entre 15 y 22px para el texto en general 
+ __Font-weight:__ usar un valor bajo para font-sizes grandes
+ __Line-height:__ entre 120 y 150% del tamaño de la letra 
+ __Caracteres por linea:__ entre 45 y 90 por linea
+ __Font-family:__ para __diseños modernos, simples y limpios:__ familia: __Sans-serif:__
  + Open sans, Lato, Raleway, Montserrat y PT Sans
  
+ __Font-family:__ para __diseños tradicionales, textos largos, artículos, blogs, libros__: familia: __Serif__
  + Cardo, Merriweather y PT Serif
  
+ Elige un __color principal__, que sea diferente de blanco, negro o gris para los elemntos __que desees que llamen la atencion de tu sitio web__
+ Cuando se coloque __texto sobre alguna imagen__, es necesario asegurarse __que se leea correctamente__, para esto puede ayudar colocar una capa sobre la imagen para resaltar el texto, o colocar un color de fondo con transparencia en el texto e incluso usar gradientes para combinar dos colores de fondo
+ __Iconos.-__ es util utilizarlos __para mostrar características, servicios y en los menus__ (si los usas en los menus, el icono debe de transmitir instantaneamente lo que deseamos y ademas es necesario colocarle a un lado su etiqueta
+ __Manejo de espacios__ entre los elementos de __cada seccion y entre secciones__
+ Define correctamente el __orden de contenido__, ya que cada espacio entre secciones muestra la relación de contenido del sitio web.
