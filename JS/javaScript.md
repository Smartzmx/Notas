# Express
Instalar express
1.- npm init (inicializar el proyecto)
2.- npm add body-parser ejs express node-fetch (para instalar las dependencias que se requieren)
  2.1.- Body parser.- transformar un string de datos a un objeto
  2.2.- ejs.- poder cargar vistas (sitios web desde el server desde el backend)
  2.3.- express.- es un servidor para hacer apis
  2.4.- node-fetch.- obtener y hacer llamadas asincronas
3.- crear un archivo dentro de la carpeta del proyecto app.js
4.- crear las siguientes carpetas:
  controllers
  models
  views
  public
5.- control + c
