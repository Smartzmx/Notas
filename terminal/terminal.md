# Comandos básicos en la terminal:

## Windows & Mac & Linux
* __mkdir__ Nos permite crear carpetas (por ejemplo, mkdir Carpeta).

* __cd__ Nos permite navegar entre carpetas.
  * __cd /__ Ir a la ruta principal:
  * __cd o cd ~__ Ir a la ruta de tu usuario
  * __cd carpeta/subcarpeta__ Navegar a una ruta dentro de la carpeta donde estamos ahora mismo.
  * __cd .. (cd + dos puntos)__ Regresar una carpeta hacia atrás.
 Si quieres referirte al directorio en el que te encuentras ahora mismo puedes usar cd . (cd + un punto).

## Mac & Linux
* __ls__ Nos permite cambiar ver los archivos de la carpeta donde estamos ahora mismo. Podemos usar uno o más argumentos para ver más información sobre estos archivos (los argumentos pueden ser -- + el nombre del argumento o - + una sola letra o shortcut por cada argumento).
  * __ls -a__ Mostrar todos los archivos, incluso los ocultos.
  * __ls -l__ Ver todos los archivos como una lista.

* __clear__ Para limpiar la terminal. También podemos usar los atajos de teclado Ctrl + L o Command + L.

* __touch__ Nos permite crear archivos (por ejemplo, touch archivo.txt).
* __echo__ anexar informacion al archivo echo "console..."> nombre del archivo
* __echo__ anexar informacion agregando info al archivo echo "console...">> nombre del archivo

* __cp ruta__ copiar un archivo a otra carpeta

* __mv nombreDelArchivo ruta__ mover un archivo a otra carpeta
* __chmod__ para modificar permisos

* __rm nombreDelArchivo__ Nos permite borrar un archivo (por ejemplo, rm archivo.txt).

* __rm -r nombreDeLaCarpeta__ borra la carpeta y todos los archivos contenidos en ésta.

* __open nombreDeLaCarpeta__ abre un archivo en específico (open nombre completo del archivo)


## Windows
* __dir__ Nos permite cambiar ver los archivos de la carpeta donde estamos ahora mismo. Podemos usar uno o más argumentos para ver más información sobre estos archivos (los argumentos pueden ser -- + el nombre del argumento o - + una sola letra o shortcut por cada argumento).

* __copy Nul nombreDelArchivo__ para crear un archivo en la terminal

* __cls__ para limpiar la terminal

* __copy ruta__ copiar un archivo a otra carpeta

* __move nombreDelArchivo ruta__ mover un archivo a otra carpeta (move nombreDelArchivo ruta)

* __rm -rf nombre del archivo__ borrar un archivo
* __rmdir nombre del archivo__ borrar un archivo

* __rmdir /S nombreDeLaCarpeta__ borrar una carpeta

* __rmdir /S nombreDeLaCarpeta__ borrar una carpeta

* __start nombreDeLaCarpeta__ abre un archivo en específico (open nombre completo del archivo)


## Especiales
* cat: Ver el contenido de un archivo (por ejemplo, cat nombre-archivo.txt).

* history: Ver los últimos comandos que ejecutamos y un número especial con el que podemos repetir su ejecución.

* ! + número: Ejecutar algún comando con el número que nos muestra el comando history (por ejemplo, !72).

* pwd: Nos muestra la ruta de carpetas en la que te encuentras ahora mismo.
