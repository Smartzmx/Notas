# Shortcuts Visual Studio Code

## Mover línea.- 
    ALT + Flecha arriba / abajo

## Mover bloque de código.- 
    Selecciono el bloque
    ALT + Flecha arriba / abajo

## Comentarios por línea.- 
    CTRL + }

## Comentario parcial de línea.- 
    SHIFT + ALT + A

## Crear carpetas / archivos de un link.- 
    CTRL + Click derecho

## Ir a la definición de una función.- 
    Click sobre la palabra + dejar sostenido CTRL

## Borrar lineas.- 
    Cursos en cualquier parte de la línea
    CTRL + SHIFT + K

## Deshacer.-
    CTRL + Z

## Re-Hacer.-
    CTRL + SHIFT + Z

## Guardar cambios.-
    CTRL + S

## Ocultar / Mostrar sidebar.-
    CTRL + B

## Ocultar / Mostrar terminal.-
    CTRL + Ñ

## Envolver una palabra en brackets de código.-
    Seleccionar la palabra
    Ctrl + SHIFT + P
    Wrap with abbreviation
    Escribir la etiqueta + Enter

## Cambiar de Tab
    CTRL + TAB

## Clonar líneas.-
    SHIFT + ALT + Flechas arriba / abajo

## Multicursor, para escribir al mismo tiempo.-
    CTRL + ALT + Flechas arriba / abajo

## Seleccionar palabras de distintas extensiones, que inican en la misma tabulación.-
    CTRL + ALT + Flechas arriba / abajo
    CRTL + SHIFT

## Transformar de mayusculas a minusculas.-
    CTRL + L

## Transformar de minusculas a Mayusculas.-
    CTRL + U

## Seleccionar la palabra completa.-
    ALT + Doble click

## Crear múltiples cursores usando el "Next find match".-
    CRTL + D

## Ir a una linea en especifico
    CRTL + G
    Escribir el número de linea deseado
