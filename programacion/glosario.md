
# Glosario de programación
**Lenguaje de programación.-** es la forma de decirle a la computadora que hacer y como hacerlo; es un idioma intermedio entre la computadora y el ser humano.

**Hardware.-** es todo lo que podemos tocar de un equipo / es la parte física de la computadora.

**Software.-** es la parte lógica, donde ocurren los procesos intrenos de los programas de la computadora.

**Lenguaje de alto nivel.-** es un lenguaje parecido al inglés o español
    Ejemplo: JavaScrip / Typescript / Java / C++ / Swift / PHP

**Lenguaje de bajo nivel.-** es un lenguaje más directo a las computadoras.
    Ejemplos: Binario / lenguaje de maquina y ensamblador.

**Plataformas.-** son los lugares donde voy a desplegar mi aplicación.

**Frameworks.-** es un conjunto de reglas o normas para la correcta utilización del lenguaje.
    Ejemplo: Angular / NodeJS / VieJS / React

**Servidor.-** es una computadora con acceso restringido y centralizado.

**Cliente.-**computadora que ejecuta el programa.

**Compilador.-** es un programa que toma todo el código que escribimos y lo traduce a un lenguaje de máquina. Los archivos .exe son el producto de la compilación del código de un programa en particular.

**Interprete.-** es un programa que  traduce y ejecuta el código linea por linea cuando se le indica a traves de algun comando.

**Lógica de programación.-** es la lógica que utiliza el programador para resolver algun problema o indicar alguna actividad a la computadora.

**Algoritmo.-** es un proceso ordenado y secuencial para resolver un problema.

**Variable.-** es el espacio en memoria donde podemos almacenar datos.

**Palabra reservada.-** son palabras que solo se pueden utilizar de la manera en que estan definidas por el lenguaje de programación.

**Comentario.-** es un bloque de código que no se ejecuta por el interprete o compilador.
